4月 23, 2025

期刊论文

  • A Landmark Study on AI
    Nature, 2023. 该论文被引用超过 1000 次，是我团队的重要里程碑。

  • Quantum Stuff Nobody Understands
    Science, 2022. 与 MIT 合作完成。

  • QTER: QoS-Aware 3D Routing for LEO Satellites
    IEEE Internet of Things Journal, 2025. 提出一种高效可靠的三维路由协议，面向 LEO 卫星网络。

  • ACDV: Adaptive Content Delivery for Vehicular Digital Twins
    IEEE Transactions on Vehicular Technology, 2025. 针对车联网数字孪生设计的自适应内容分发策略。

  • Digital Twin in 6G: Toward Network Intelligence
    IEEE Wireless Communications, 2024. 与多所高校合作，为 6G 网络智能化注入数字孪生理念。

  • Game-Theoretic Data Sync in Vehicular Twins
    IEEE Transactions on Wireless Communications, 2023. 引入博弈论方法，实现车联网中数字孪生的数据同步。

  • Learning-Based Task Offloading for IoVs
    IEEE Transactions on Network Science and Engineering, 2023. 利用机器学习优化车联网中的任务卸载。

  • Heterogeneous Network Selection with Knowledge Transfer
    IEEE Transactions on Vehicular Technology, 2022. 借助知识迁移，提升异构车载网络中的选择效率。

  • SECR: Secure Charging Reservation in Vehicular Twins
    IEEE Internet of Things Journal, 2024. 设计安全高效的充电预约机制，适配车载数字孪生系统。

  • Lightweight Data Sharing in Vehicular Twins
    Peer-to-Peer Networking and Applications, 2024. 面向车联网提出轻量级安全数据共享方案。

  • Smart City Data Sharing via Digital Twins
    IEEE Network, 2023. 聚焦智慧城市，通过数字孪生打破数据壁垒。

  • AMIS-MU: Adaptive Video Streaming for Mobile Users
    IEEE Transactions on Mobile Computing, 2022. 基于边缘计算，为多移动用户优化视频流服务。

  • ROS-Based Collaborative Driving Framework
    IEEE Transactions on Vehicular Technology, 2023. 基于 ROS 框架，支持自动驾驶中的协同驾驶机制。

会议论文

  • Content Delivery via Heterogeneous Networks in Vehicular Twins
    IEEE ICC, 2025. 基于异构网络，优化车联网数字孪生内容传输性能。

  • Adaptive Multi-Link Allocation for LEO Satellites
    IEEE GlobeCom, 2024. 提出多链路自适应数据分配机制，面向 LEO 卫星网络。

  • Long-Term Incentives for Federated Learning
    IEEE PIMRC, 2023. 利用动态重复博弈，激励联邦学习中的长期合作。

  • RL-Based Device Placement for Autonomous Vehicles
    IEEE CPSCom, 2021. 应用强化学习优化自动驾驶车辆设备部署策略。

  • AMIS: Adaptive Mobile Video via Edge Computing
    IEEE INFOCOM, 2021. 借助边缘计算，实现移动视频自适应流媒体传输。

  • Data Synchronization for Vehicular Twins
    IEEE GlobeCom, 2022. 聚焦车联网数字孪生的数据同步机制。

  • Data Asset Protection in Metaverse: A Game Theoretic Method
    IEEE MetaCom, 2023. 荣获最佳论文奖，提出元宇宙数据资产保护博弈模型。

  • Secure Handover Authentication for 5G-V2X
    IEEE GlobeCom, 2023. 结合数字孪生技术，提升 5G-V2X 切换安全性。

  • Secure Charging Reservation for EVs via Digital Twin
    IEEE ICCC, 2023. 在电动车充电预约中引入安全的数字孪生机制。

  • Incentive Scheme for Data Sync in Vehicular Twins
    IEEE VTC-Spring, 2024. 采用 Stackelberg 博弈模型，激励车联网中数据同步。

  • Nightfall Deception: Backdoor Attack via Low-Light Manipulation
    ADMA, Springer, 2024. 创新性提出低光照数据操控的交通标识后门攻击。

  • Semi-Tensor Sparse Coding for Short-Packet URLLC
    IEEE WCNC, 2025. 面向 URLLC 提出低存储开销的稀疏向量编码方案。

  • Stackelberg-Based Resource Trading in DAG-MEC Networks
    IEEE WCNC, 2025. 结合 DAG 区块链与 Stackelberg 博弈，优化 MEC 资源交易。
